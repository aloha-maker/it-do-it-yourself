# リポジトリオーナーのみ
 - ローカルでリポジトリと同じフォルダを作成し、"README.md"ファイルを作成する。
 - 下記のコマンドを実行する。
   - 説明：
     - git管理を始める
     - README.md　をアップロードする
     - mainブランチにする
     - ローカルとリモートをつなぐ

```bash
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/{$ユーザー名}/{$リポジトリ名}.git #URLはGitHubからコピー
git push -u origin main
```
